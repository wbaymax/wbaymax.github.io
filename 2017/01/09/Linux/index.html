pscp -r user@ip:/remote_dir \local_dir
```
例：使用dodo用户将Linux主机中/linuxidc目录以及目录中的文件保存到本地
pscp -r dodo@192.168.100.250:/linuxidc/hello1 D:\hello2
这样就会把hello1目录保存在hello2目录下
根目录下的目录hello1
